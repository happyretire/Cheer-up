# 📣 Cheer-up | 세련된 전광판 메신저

고등학생부터 성인까지, 응원이 필요한 모든 순간에 사용할 수 있는 **세련된 감성의 웹 전광판**입니다. 
복잡한 설치 없이 브라우저에서 바로 실행하여 힙한 디자인의 메시지를 띄울 수 있습니다. ✨

---

## ✨ 주요 특징

- **세련된 타이포그래피**: 가독성과 감성을 모두 잡은 '고운 돋움' 폰트 적용
- **동적 오로라 배경**: 은은하게 움직이는 파스텔톤 배경으로 고급스러운 분위기 연출
- **실시간 메시지 변경**: 하단 제어판에서 원하는 문구를 즉시 입력하고 적용 가능
- **상황별 추천 태그**: 수험, 입학, 축제, 졸업 등 상황에 맞는 힙한 문구 원클릭 적용
- **제어 기능**: 문구 흐름 일시 정지(STOP/PLAY) 및 제어판 숨기기 지원

## 🚀 사용 방법

1. `index.html` 파일을 더블 클릭하여 브라우저에서 엽니다.
2. 화면 하단의 입력창에 응원 메시지를 입력하고 **[적용]** 버튼을 누릅니다. (엔터 키 가능)
3. **[STOP/PLAY]** 버튼으로 글자를 멈추거나 다시 움직이게 할 수 있습니다.
4. 상황별 버튼(#수험, #입학 등)을 눌러 추천 문구를 바로 사용할 수 있습니다.
5. 우측 상단의 **⚙️ 아이콘**을 누르면 입력창을 숨겨 전광판을 깔끔하게 볼 수 있습니다.

## 💡 추천 활용 상황

- **학교 행사**: 축제 무대 응원, 체육대회 반별 응원
- **수험 생활**: 자습실 문구, 시험 기간 친구 응원
- **기념일**: 입학, 졸업, 생일 축하 메신저
- **일상**: 오늘 점심 메뉴 공지, 가벼운 장난 문구
